My blog
=======

Code source for my blog.

I'm using [minimal-mistakes](https://github.com/mmistakes/minimal-mistakes)
Jekyll theme.


## Install

```
git clone git@github.com:alcalyn/alcalyn.github.io.git
cd alcalyn.github.io/

docker-compose up
```

Then go to http://0.0.0.0:4000
